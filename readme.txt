Simple responsive tabs for WordPress with awesome icons

== Description ==

Organize any content of your WordPress website with these convenient responsive tabs. Ready for both mobile and desktop browsing, these tabs are extremely easy to install, use and configure. Assign icons to each tab, use vertical or horizontal layout, and deploy as many tabs or set of tabs as you need in a few clicks. 
Moreover, we've included simple integration of these tabs with our interactive maps, so with Tabs Maker plugin you can effortlessly embed [free](https://wordpress.org/plugins/html5-maps/) and [premium interactive responsive maps](https://www.fla-shop.com/wordpressmaps.php) directly to the tabs. Or use shortcodes of any third-party plugins to display arbitrary content.

### Features

*	Responsive design adapts to any resolution or browser size
*	Works great on mobile and desktop browsers
*	Horizontal and vertical layout come as standard
*	Simple visual configuration including awesome tab icons
*	Easy to add tabs to a page via shortcodes

== Installation ==

1. In your WordPress dashboard panel, go to Plugins > Add new
2. Find our **Tabs Maker** plugin by Fla-shop.com and click Install now
3. Wait for the installation to complete and activate the plugin.

After activate plugin you will see "Tabs Maker" menu at left side on WordPress dashboard. Click "Add new tab set" to create a new set of tabs.
